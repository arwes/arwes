# Contributing to the website

> The website application is still under development.

The [website](https://arwes.dev) is a [Gatsby](https://gatsbyjs.com) web
application written in JavaScript.

## Setting up

The repository has to be set up first. See [workflow instructions](/project/contributing/workflow).

Then the website can be installed:

```bash
# at root folder
cd website
npm install
```

## Running

```bash
# at root folder
cd website
npm run develop
```
